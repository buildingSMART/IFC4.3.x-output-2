E.4.10 - Slab Extrusion

Example overview

﻿Additional test cases not yet allocated to individual test case groups.

﻿This example illustrates a standard-case slab with extruded solid geometry, based on a material layer set usage definition. Figure A shows the resulting shape.

NOTE  The extruded profile is defined by IfcIndexedPolyCurve
NOTE  There is no color information within the file, the displayed color has been set by the target application as a default.
