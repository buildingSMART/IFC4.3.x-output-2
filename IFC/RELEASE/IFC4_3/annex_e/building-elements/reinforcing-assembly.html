E.4.8 - Reinforcing Assembly

Example overview

﻿Additional test cases not yet allocated to individual test case groups.

﻿This example illustrates a beam with an assembly of reinforcing stirrups, the geometry of the stirrups is defined by mapped items linking to a mapped representation being a swept disk solid along a directrix. Figure A shows the resulting shape.

NOTE  The directrix is defined by IfcIndexedPolyCurve
NOTE  There is no color information within the file, the displayed color has been set by the target application as a default.
