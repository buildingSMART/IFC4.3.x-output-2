E.4.3 - Beam Revolved Solid

Example overview

﻿Additional test cases not yet allocated to individual test case groups.

﻿This example illustrates a beam with revolved solid geometry, based on an parametric I-Shape profile and corresponding material profile set usage definition. Figure A shows the resulting shape.

NOTE  There is no color information within the file, the displayed color has been set by the target application as a default.

IFC-SPF source
