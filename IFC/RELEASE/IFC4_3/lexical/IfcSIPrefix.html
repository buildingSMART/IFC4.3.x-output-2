8.11.2.82 IfcSIPrefix

8.11.2.82.1 Semantic definition

{ .extDef}

NOTE Definition according to ISO/CD 10303-41:1992 An SI prefix is the name of a prefix that may be associated with an SI unit. The definitions of SI prefixes are specified in ISO 1000 (clause 3).
NOTE The prefix defines multiples and submultiples of the SI units.
NOTE Type adapted from si_prefix defined in ISO 10303-41.
HISTORY New entity in IFC1.5.1.
