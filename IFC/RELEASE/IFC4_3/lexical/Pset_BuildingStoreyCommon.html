Propriedades comuns para a definição de todas as instâncias de IfcBuildingStorey. Por favor, note que vários atributos de edifício são tratados diretamente na instância IfcBuildingStorey, o número de andares de construção (ou nome curto) por IfcBuildingStorey.[Name]], o nome do andar do edifício (ou nome longo) por IfcBuildingStorey.[LongName]], e a descrição (ou comentários) por IfcBuildingStorey.Description. IfcBuildingStorey’in tüm örneklerinin tanımında ortak olan özellikler. Lütfen dikkat edin: bazı bina katı nitelikleri doğrudan IfcBuildingStorey örneği tarafından ele alınır; kat numarası (veya kısa adı) IfcBuildingStorey.Name, kat adı (veya uzun adı) IfcBuildingStorey.LongName ve açıklama (veya yorumlar) IfcBuildingStorey.Description tarafından ele alınır. Gerçek bina katı miktarları, örneğin bina katı çevresi, bina katı alanı ve bina katı hacmi, IfcElementQuantity tarafından sağlanır. Kat sınıflandırması, ulusal bina koduna göre IfcClassificationReference tarafından sağlanır.

5.4.4.8.1 Semantic definition

Properties common to the definition of all instances of IfcBuildingStorey. Please note that several building attributes are handled directly at the IfcBuildingStorey instance, the building storey number (or short name) by IfcBuildingStorey.Name, the building storey name (or long name) by IfcBuildingStorey.LongName, and the description (or comments) by IfcBuildingStorey.Description. Actual building storey quantities, like building storey perimeter, building storey area and building storey volume are provided by IfcElementQuantity, and the building storey classification according to national building code by IfcClassificationReference.
